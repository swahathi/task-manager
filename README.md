# Simple Task Manager

## Description

A simple web-based Task Manager developed using HTML, CSS, and JavaScript. The application allows users to add, complete, and delete tasks easily.

## Features

* Add new tasks
* Add tasks using the Enter key
* Mark tasks as completed
* Delete tasks
* Clean and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```
task-manager/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Start managing your tasks.
